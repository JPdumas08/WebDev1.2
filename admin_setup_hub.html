Setup Instructions

Follow these steps to get the admin dashboard fully functional:

Method 1: Web-Based Setup (Easiest)

  1. Click the "Setup Database" button above
  2. Wait for the setup to complete (should see success message)
  3. Run the "Run Diagnostic" to verify
  4. Click "Go to Admin" to access the dashboard
  5. Log in with your admin account
